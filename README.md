# CRM Opportunity Dashboard

## Project Overview

This project presents an interactive CRM Opportunity Dashboard built using Power BI and Tableau to monitor sales opportunities, revenue forecasting, and pipeline performance. The dashboard helps stakeholders analyze opportunity conversion, win rates, expected revenue, and industry-wise business performance.

## Tools Used

- Power BI
- Tableau
- Microsoft Excel
- Power Query
- DAX

## Skills Demonstrated

- Dashboard Design
- Data Visualization
- Sales Analytics
- KPI Reporting
- DAX
- Power Query
- Business Intelligence

## Dashboard Features

- Expected Amount Analysis
- Forecast vs Expected Revenue
- Active Opportunities
- Conversion Rate
- Win Rate
- Industry Analysis
- Closed Won Opportunities
- Opportunity Pipeline

## Key Insights

- Monitor sales pipeline performance.
- Compare forecasted and expected revenue.
- Track conversion and win rates.
- Identify high-value industries.
- Analyze opportunity distribution.
